# M3u8

帮助你下载 `.m3u8` 文件承载的媒体内容, 并做好相关记录防止重复下载。

使用 `SQLite` 本地数据库做下载记录存储。

使用 `ffmpeg` 下载 `.m3u8` 媒体并做转码合并。因此需要你先在本地下载好 ffmpeg 并将其所在目录加入到系统 `Path` 环境变量中。

默认保存在 用户的下载文件夹 下的 .m3u8 文件夹中, `%UserProfile%/Downloads/.m3u8`

## 如何使用

1. 首次使用需要先执行 `init` 子命令初始化数据库。单独做初始化的目的是提高程序的执行效率。

1. 使用 `add` 子命令添加你需要管理的 m3u8 的 url, 或者是保存了大量 m3u8 链接的 .txt 文本文档。简而言之就是单个链接直接填, 多个链接则从文本文档中读取(一个链接一行)。

1. 使用 `download` 子目录执行下载, 后面可以接一个数字表示你需要下载的数量, 默认是1。
